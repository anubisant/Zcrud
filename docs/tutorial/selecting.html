Introduction

All configuration option about selecting is set in selecting component. For lists these options are pageConf/pages/list/components/selecting.

Selecting rows

Selecting rows is off by default. Default configuration for lists is: 

        pageConf: {
            pages: {
                list: {
                    components: {
                        ...
                        selecting: {
                            isOn: false,
                            constructorClass: require( './components/selectingComponent.js' ),
                            multiple: true,
                            mode: [ 'checkbox', 'onRowClick' ] // possible values: 'checkbox' and 'onRowClick'
                        },
                        ...
                    }
                }
            }
        }

Let's activate selecting support: 

        pageConf: {
            pages: {
                list: {
                    components: {
                        selecting: {
                            isOn: true
                        }
                    }
                }
            }
        }

To get an array with the selected records use the getSelectedRecords method: 

             var selectedRecords = $( '#container' ).zcrud( 'getSelectedRecords' );

To get a jquery selection with the selected rows use the getSelectedRows method: 

             var $selectedRows = $( '#container' ).zcrud( 'getSelectedRows' ):
